# Emotion의 css prop

Emotion에서 가장 기본적인 스타일링 방법

## css prop이란?

`css` prop은 Emotion이 제공하는 JSX 요소에 직접 스타일을 적용하는 방법이다.
인라인 스타일과 비슷해 보이지만, 실제 CSS 클래스가 생성되어 적용된다.

## 설정 방법

### 방법 1: jsxImportSource pragma 사용

```jsx
/** @jsxImportSource @emotion/react */
import { css } from '@emotion/react';

function Button() {
  return (
    <button
      css={css`
        background: blue;
        color: white;
      `}
    >
      Click me
    </button>
  );
}
```

### 방법 2: Babel 설정

```json
// .babelrc 또는 babel.config.json
{
  "presets": [
    [
      "@babel/preset-react",
      { "runtime": "automatic", "importSource": "@emotion/react" }
    ]
  ],
  "plugins": ["@emotion/babel-plugin"]
}
```

## 사용 패턴

### 1. 템플릿 리터럴 방식

```jsx
<div
  css={css`
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
  `}
>
  Centered Content
</div>
```

### 2. 객체 스타일 방식

```jsx
<div
  css={{
    display: 'flex',
    justifyContent: 'center',
    alignItems: 'center',
    height: '100vh',
  }}
>
  Centered Content
</div>
```

### 3. 동적 스타일링

```jsx
function Box({ isActive }) {
  return (
    <div
      css={css`
        padding: 16px;
        background: ${isActive ? 'green' : 'gray'};
        color: white;
        transition: background 0.3s ease;
      `}
    >
      {isActive ? 'Active' : 'Inactive'}
    </div>
  );
}
```

### 4. 스타일 재사용

```jsx
const baseStyle = css`
  padding: 16px;
  border-radius: 8px;
  font-size: 14px;
`;

const primaryStyle = css`
  ${baseStyle}
  background: blue;
  color: white;
`;

const secondaryStyle = css`
  ${baseStyle}
  background: gray;
  color: black;
`;

function Buttons() {
  return (
    <>
      <button css={primaryStyle}>Primary</button>
      <button css={secondaryStyle}>Secondary</button>
    </>
  );
}
```

## 스타일 합성 (Composition)

여러 스타일을 배열로 합성할 수 있다:

```jsx
const baseButton = css`
  padding: 8px 16px;
  border: none;
  border-radius: 4px;
  cursor: pointer;
`;

const dangerButton = css`
  background: red;
  color: white;
`;

function DangerButton() {
  return <button css={[baseButton, dangerButton]}>Delete</button>;
}
```

## 주의사항

1. **css prop을 사용하려면 반드시 설정이 필요하다** (pragma 또는 Babel)
2. 객체 스타일에서는 **camelCase**를 사용한다 (backgroundColor, fontSize 등)
3. 템플릿 리터럴에서는 일반 CSS처럼 **kebab-case**를 사용한다

## 언제 사용하면 좋을까?

- 빠르게 프로토타이핑할 때
- 간단한 조건부 스타일링이 필요할 때
- 스타일이 해당 컴포넌트에서만 사용될 때
