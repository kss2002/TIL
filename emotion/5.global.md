# Emotion Global Styles

> 전역 스타일과 CSS 리셋 적용하기

## Global 컴포넌트

`@emotion/react`의 `Global` 컴포넌트를 사용하면 전역 CSS를 적용할 수 있다.

## 기본 사용법

```jsx
import { Global, css } from '@emotion/react';

function App() {
  return (
    <>
      <Global
        styles={css`
          * {
            margin: 0;
            padding: 0;
            box-sizing: border-box;
          }

          body {
            font-family:
              'Pretendard',
              -apple-system,
              BlinkMacSystemFont,
              sans-serif;
            line-height: 1.6;
            background-color: #f5f5f5;
          }

          a {
            text-decoration: none;
            color: inherit;
          }
        `}
      />
      <MainContent />
    </>
  );
}
```

## CSS Reset 적용

### 간단한 리셋

```jsx
const resetStyles = css`
  *,
  *::before,
  *::after {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
  }

  html {
    font-size: 16px;
    -webkit-font-smoothing: antialiased;
    -moz-osx-font-smoothing: grayscale;
  }

  body {
    min-height: 100vh;
    line-height: 1.5;
  }

  img,
  picture,
  video,
  canvas,
  svg {
    display: block;
    max-width: 100%;
  }

  input,
  button,
  textarea,
  select {
    font: inherit;
  }

  button {
    cursor: pointer;
    border: none;
    background: none;
  }

  ul,
  ol {
    list-style: none;
  }

  a {
    text-decoration: none;
    color: inherit;
  }
`;

function App() {
  return (
    <>
      <Global styles={resetStyles} />
      <MainContent />
    </>
  );
}
```

## 테마와 함께 사용

`Global` 컴포넌트의 `styles` prop에 함수를 전달하면 테마에 접근할 수 있다:

```jsx
import { Global, css, ThemeProvider } from '@emotion/react';

const globalStyles = (theme) => css`
  body {
    background-color: ${theme.colors.background};
    color: ${theme.colors.text};
    font-family: ${theme.fonts.body};
  }

  ::selection {
    background-color: ${theme.colors.primary};
    color: white;
  }

  ::-webkit-scrollbar {
    width: 8px;
  }

  ::-webkit-scrollbar-thumb {
    background-color: ${theme.colors.primary};
    border-radius: 4px;
  }
`;

function App() {
  return (
    <ThemeProvider theme={theme}>
      <Global styles={globalStyles} />
      <MainContent />
    </ThemeProvider>
  );
}
```

## 전역 스타일 분리하기

실제 프로젝트에서는 전역 스타일을 별도 파일로 분리하는 것이 좋다:

```jsx
// styles/GlobalStyles.jsx
import { Global, css } from '@emotion/react';

const resetStyles = css`
  /* CSS Reset */
  *,
  *::before,
  *::after {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
  }
`;

const baseStyles = (theme) => css`
  html {
    font-size: 16px;
  }

  body {
    font-family: 'Pretendard', sans-serif;
    background-color: ${theme.colors.background};
    color: ${theme.colors.text};
    line-height: 1.6;
  }
`;

const typographyStyles = css`
  h1 {
    font-size: 2.5rem;
    font-weight: 700;
    margin-bottom: 1rem;
  }

  h2 {
    font-size: 2rem;
    font-weight: 600;
    margin-bottom: 0.875rem;
  }

  h3 {
    font-size: 1.5rem;
    font-weight: 600;
    margin-bottom: 0.75rem;
  }

  p {
    margin-bottom: 1rem;
  }
`;

export function GlobalStyles() {
  return (
    <>
      <Global styles={resetStyles} />
      <Global styles={baseStyles} />
      <Global styles={typographyStyles} />
    </>
  );
}
```

```jsx
// App.jsx
import { ThemeProvider } from '@emotion/react';
import { GlobalStyles } from './styles/GlobalStyles';
import theme from './theme';

function App() {
  return (
    <ThemeProvider theme={theme}>
      <GlobalStyles />
      <MainContent />
    </ThemeProvider>
  );
}
```

## @font-face 적용

커스텀 폰트를 전역으로 적용할 수 있다:

```jsx
const fontStyles = css`
  @font-face {
    font-family: 'Pretendard';
    src: url('/fonts/Pretendard-Regular.woff2') format('woff2');
    font-weight: 400;
    font-style: normal;
    font-display: swap;
  }

  @font-face {
    font-family: 'Pretendard';
    src: url('/fonts/Pretendard-Bold.woff2') format('woff2');
    font-weight: 700;
    font-style: normal;
    font-display: swap;
  }
`;

function App() {
  return (
    <>
      <Global styles={fontStyles} />
      <MainContent />
    </>
  );
}
```

## CSS 변수 활용

CSS 변수를 전역으로 설정하고 Emotion과 함께 사용할 수 있다:

```jsx
const cssVariables = css`
  :root {
    --color-primary: #3498db;
    --color-secondary: #2ecc71;
    --spacing-sm: 8px;
    --spacing-md: 16px;
    --spacing-lg: 24px;
  }

  [data-theme='dark'] {
    --color-primary: #5dade2;
    --color-secondary: #58d68d;
  }
`;

// styled 컴포넌트에서 사용
const Button = styled.button`
  background-color: var(--color-primary);
  padding: var(--spacing-sm) var(--spacing-md);
`;
```

## 주의사항

1. **Global 컴포넌트는 렌더링될 때마다 스타일을 주입**하므로 성능에 유의
2. 가능하면 **정적인 스타일**을 Global로 사용
3. 조건부 전역 스타일이 필요하면 **CSS 변수**를 활용하는 것이 더 효율적
