# Emotion 소개

> CSS-in-JS 라이브러리의 대표주자, Emotion에 대해 알아보기

## Emotion이란?

Emotion은 JavaScript로 CSS 스타일을 작성할 수 있게 해주는 **CSS-in-JS 라이브러리**다. React와 함께 사용되는 경우가 많지만, 프레임워크에 구애받지 않고 사용할 수 있다.

## 왜 Emotion을 사용하는가?

### 1. 컴포넌트 기반 스타일링

- 스타일이 컴포넌트와 함께 존재하므로 관리가 쉽다
- 스타일 충돌 걱정 없이 개발할 수 있다

### 2. 동적 스타일링

- props나 state에 따라 스타일을 쉽게 변경할 수 있다
- 조건부 스타일링이 간단하다

### 3. 자동 벤더 프리픽싱

- 브라우저 호환성을 위한 접두사가 자동으로 추가된다

### 4. SSR (Server-Side Rendering) 지원

- Next.js 등과 함께 사용할 때 SSR을 지원한다

## 설치

```bash
# @emotion/react - React와 함께 사용할 때
npm install @emotion/react

# @emotion/styled - styled-components와 유사한 API 사용 시
npm install @emotion/styled
```

## 기본 예제

```jsx
/** @jsxImportSource @emotion/react */
import { css } from '@emotion/react';

function App() {
  return (
    <div
      css={css`
        background-color: #f0f0f0;
        padding: 20px;
        border-radius: 8px;
      `}
    >
      Hello Emotion!
    </div>
  );
}
```
