## 404 페이지

루비 온 레일즈에서도 404 페이지를 세팅할 수 있습니다.

```html
## 경로: public/404.html
<!DOCTYPE html>
<html lang="ko">
  <head>
    <title>존재하지 않는 페이지입니다</title>

    <meta charset="utf-8" />
    <meta name="viewport" content="initial-scale=1, width=device-width" />
    <meta name="robots" content="noindex, nofollow" />

    <style>
      { // 관련 인라인 css 구현 }
    </style>
  </head>

  <body>
    <!-- This file lives in public/404.html -->
    <main>
      <header>
        <!-- svg .. -->
      </header>
      <article>
        <p>홈으로 돌아가기</p>
      </article>
    </main>
  </body>
</html>
```
