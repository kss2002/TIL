# Claude Cowork

**Claude Code의 힘을 일반 지식 노동(knowledge work)으로 확장한 제품.** 코딩이 아닌 문서·데이터·파일 기반 업무를 데스크톱에서 자율적으로 처리한다. 목표만 주면 Claude가 내 컴퓨터의 파일·앱을 오가며 완성된 결과물을 만들어 온다.

## 핵심 요소

- **데스크톱에서 동작** — 로컬 파일·폴더·앱 위에서 직접 읽고, 편집하고, 새 파일을 생성 (사용자가 지정한 폴더 범위 내)
- **자율 멀티스텝** — 여러 소스의 정보를 종합해 단계별 지시 없이 작업을 끝까지 수행
- **주요 기능** — Connectors, Plugins, Scheduled Tasks(예약 작업), Dispatch, Projects, Computer Use
- **타깃 사용자** — 엔지니어가 아닌 리서처·분석가·운영·법무·재무 등 문서/데이터를 다루는 모든 팀

## 타임라인

- 2026년 1월 12일: 리서치 프리뷰로 공개 (비개발자용 "컴퓨터 에이전트")
- 이후 모든 유료 플랜으로 **정식 출시(GA)**
