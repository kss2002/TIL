# Claude for Team / Enterprise

개인 플랜(Free·Pro·Max)을 넘어 **조직 단위로 Claude를 도입**하는 플랜. 관리자 제어, 보안/규정 기능, 더 큰 컨텍스트가 핵심 차이다.

## Team

- 최소 **5석(seat)**부터
- **Standard**: $25/석/월 (연간 결제 시 $20)
- **Premium**: $125/석/월 (연간 $100) — Standard의 5배 사용량
- 포함: SAML SSO, 중앙 청구, 관리자 제어, Microsoft 365 연동, 기본적으로 팀 대화 모델 학습 안 함

## Enterprise

- 맞춤 가격(약 $20/석/월부터), **API 사용량은 별도 청구** — 채팅·Claude Code·Cowork에서 쓴 토큰이 표준 API 요율로 부과
- **500K 컨텍스트 윈도우** (Team은 200K)
- HIPAA 대응, SCIM, 감사 로그(audit log), SSO

## 요약

| | Team | Enterprise |
|---|---|---|
| 컨텍스트 | 200K | 500K |
| 보안 | SSO, 관리자 제어 | + SCIM, 감사 로그, HIPAA |
| 가격 | 공개 ($20~125/석) | 맞춤 (영업 문의) |
