# Claude 앱 (claude.ai)

가장 기본이 되는 **대화형 Claude 제품.** 웹·데스크톱·모바일에서 채팅으로 사용하며, Projects와 Artifacts가 핵심 기능이다.

## 핵심 요소

### Artifacts
대화 옆 별도 패널에 **독립적인 결과물**을 만들어 주고, 즉시 미리보기·수정할 수 있는 기능.

- React 앱, HTML 페이지, SVG, Mermaid 다이어그램, 코드, 마크다운을 샌드박스에서 실행
- MCP로 외부 서비스(Asana, Google Calendar, Slack 등)와 연동 가능
- 세션 간 데이터 저장 → 저널·트래커 같은 상태 유지 앱도 제작
- 현재 모든 유료 플랜 + Projects 안에서 사용 가능

### Projects
관련 채팅과 문서를 한 곳에 모으는 **격리된 작업 공간.**

- 참고 문서·시스템 지시를 업로드하면 대화 전반에 걸쳐 컨텍스트 유지
- 전체를 통째로 넣는 게 아니라 관련 부분만 검색해 가져오는 방식
- 프로젝트당 **200K 컨텍스트 윈도우**(약 500페이지 분량)
