## 🗂️ 전체 CS를 한 번에 다루는 곳

**[gyoogle/tech-interview-for-developer](https://github.com/gyoogle/tech-interview-for-developer)**
GitHub에서 가장 유명한 한국어 CS 정리 레포지토리입니다. 디자인 패턴, 네트워크, OS, DB, 자료구조를 모두 마크다운으로 정리해 두었고, 기술 면접 기준으로 핵심만 담겨 있어서 빠르게 훑기에 좋습니다.

**[Seogeurim/CS-study](https://github.com/Seogeurim/CS-study)**
자료구조, 알고리즘, 운영체제, 데이터베이스, 네트워크, 디자인 패턴, 소프트웨어 공학까지 컴공 전공자로서 알아야 할 지식을 이론부터 구현까지 정리한 GitHub 스터디 저장소입니다.

**[TeachYourselfCS 한국어 번역](https://github.com/minnsane/TeachYourselfCS-KR)**
독학 엔지니어, 부트캠프 졸업생을 염두에 두고 만들어진 가이드로, 학습 추천 순서와 각 과목에 어떤 교재/강의를 활용할지까지 안내해 줍니다. 영어 원문 사이트의 번역본입니다.

**[인프런 - 향로의 CS 전공 로드맵](https://www.inflearn.com/roadmaps/1641)**
컴퓨터구조, 운영체제, 알고리즘, 자료구조, 네트워크, 데이터베이스를 모두 다루는 인프런 공식 로드맵으로, 각 과목별로 가장 친절한 강의들만 큐레이션해 두었습니다. 로드맵 자체는 무료로 볼 수 있고, 연결된 강의 중 일부는 무료입니다.

## 📌 파트별 추천 (전부 무료)

### 운영체제

**[KOCW - 반효경 교수님 운영체제 (이화여대)](http://www.kocw.net/home/search/kemView.do?kemId=1046323)**
운영체제의 개념과 역할, 각 구성 요소 및 알고리즘의 핵심을 기초부터 배울 수 있는 강의로, 국내에서 가장 유명한 무료 OS 강의입니다. 완강 후기가 넘치는 명강으로 꼽힙니다.

추가로 **[인프런 - 주니온 교수님의 운영체제 공룡책 강의](https://www.inflearn.com/course/운영체제-공룡책-전공강의)** 도 무료입니다.

### 네트워크

**[KOCW - 이석복 교수님 컴퓨터네트워크 (한양대)](http://www.kocw.net/home/search/kemView.do?kemId=1169634)** 가 가장 많이 추천되는 무료 강의입니다.

### 데이터베이스

**[KOCW - 백현미 교수님 (한양대)](http://www.kocw.net/home/search/kemView.do?kemId=1163794)** 과 **[KOCW - 신주현 교수님 (조선대)](http://www.kocw.net/home/search/kemView.do?kemId=1385412)** 가 있으며, 둘 다 무료로 수강 가능합니다.

### 자료구조

**[KOCW - 권오흠 교수님 자료구조 및 알고리즘](https://www.youtube.com/channel/UC-cOmaeWLm7Ii7erMQNatvA)** 유튜브 채널에서 무료로 시청 가능합니다.

### 디자인 패턴

별도의 무료 강의는 적은 편이고, 위에서 소개한 GitHub 레포들(gyoogle, Seogeurim)의 디자인 패턴 파트를 읽는 방식이 가장 현실적입니다. 좀 더 깊이 보고 싶다면 **[Refactoring.Guru](https://refactoring.guru/ko/design-patterns)** 사이트가 한국어를 지원하고 그림으로 잘 설명되어 있어서 추천드립니다.
